# CATS! – Facts & Chaos

**CATS! – Facts & Chaos** is a mobile-first, responsive interactive website created purely for fun.  
The project combines random cat facts, cute images, sound effects, and a bit of intentional chaos.

## Current Features

- Cute cat images fetched from a public API
- Interactive “trolling” button (DEFINITELY WITHOUT CONFETTI)
- Start screen inspired by *BroForce*
- Mobile-first responsive layout

## Planned Features

- Fix issues with loading and displaying facts
- Add more sound effects
- Enhance project documentation (README)
